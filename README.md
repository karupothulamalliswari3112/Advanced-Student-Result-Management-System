# Advanced Student Result Management System

## 📌 Description
This project is an Advanced Student Result Management System developed using Python. It allows users to add, view, and store student results efficiently using file handling techniques.

## 🚀 Features
* Add student details
* View student results
* Save results to file
* Simple and user-friendly console interface

## 🛠 Technologies Used
* Python
* File Handling
* Data Structures (List, Dictionary)

## ▶️ How to Run
1. Open terminal
2. Navigate to project folder
3. Run the command:
   python student_result.py

## 📂 Output
* The results are saved in a file named `results.txt`
